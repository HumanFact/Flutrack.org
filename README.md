Flutrack.org
============

Flutrack platform gathers flu related tweets from the entire world, with searching tag, words that are influenza synonyms and flu symptoms. The tags that being tracked are: Influenza, flu, chills, headache, sore throat, runny nose, sneezing, fever, dry cough. For every tweet extracted, additional amount of metadata is extracted too.
